# fastInitGulp
Repository for fast init Gulp project with pre-installed packages
 
# Initialization

Install the dependencies and devDependencies and start the server.
```sh
$ npm i
$ gulp
```

## For production

To build the project, write:

```$ gulp build```

## Using technologies:
  - gulp
  - SASS
  - jade (Pug)
